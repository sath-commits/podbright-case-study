# Assets

Place screenshots here before publishing.

## Recommended screenshots

| File | Description |
|---|---|
| `dashboard.png` | Dashboard with 5–8 episodes in various states (ready, processing, queued) |
| `episode-card.png` | Single episode card showing chapters, duration, play button |
| `settings.png` | Settings page showing voice selector + mode/difficulty/duration options |
| `onboarding.png` | Onboarding flow showing feed URL + inbox address |
| `mobile-episodes.png` | Mobile app episode list |
| `mobile-settings.png` | Mobile app settings with all preference pickers |

## Screenshot guidelines

- Dark background (matches site theme)
- Real or realistic-looking content (no "test episode 1", "foo bar")
- Retina resolution (minimum 1200px wide for desktop shots)
- No browser chrome unless intentional
- Crop to the relevant UI — no desktop background visible
